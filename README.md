# limit-music

## 官网地址: http://43.138.37.12/limitMusic

### 这是一个使用Electron 开发的桌面端app

### Install / 安装

* 推荐使用pnpm进行安装

```bash
$ pnpm install
```

### Development / 开发

```bash
$ pnpm dev
```

### Build / 打包

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```

### 外置控制器调试

```bash
# 调试前执行
$ pnpm build-web
```

### 控制器页面开发


```bash
$ pnpm dev-web
```
